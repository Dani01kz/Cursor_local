# Cursor — Быстрый старт

## Что такое Cursor?

Cursor — это AI-редактор кода на базе VS Code. Он поддерживает автодополнение, чат с ИИ и автоматическое редактирование кода.

## Как подключиться к Cursor

### 1. Установка

1. Скачайте Cursor с официального сайта: [cursor.com](https://www.cursor.com/)
2. Установите приложение (доступно для Windows, macOS, Linux)
3. Запустите Cursor

### 2. Авторизация

При первом запуске Cursor предложит войти в аккаунт:

- Нажмите **Sign In**
- Войдите через GitHub, Google или email
- Готово — вы в Cursor

### 3. Открытие проекта

- **File → Open Folder** — выберите папку с проектом
- Или из терминала: `cursor /path/to/project`

### 4. Подключение к Git-репозиторию

```bash
# Клонируйте репозиторий
git clone https://github.com/username/repo.git

# Откройте его в Cursor
cursor repo/
```

Cursor автоматически обнаружит Git и покажет изменения во вкладке Source Control.

### 5. Использование AI-функций

| Действие | Горячая клавиша |
|---|---|
| Открыть AI-чат | `Ctrl+L` (или `Cmd+L` на Mac) |
| Inline-редактирование | `Ctrl+K` (или `Cmd+K` на Mac) |
| Автодополнение | `Tab` (принять) / `Esc` (отклонить) |

### 6. Настройки (опционально)

- **Cursor Settings** (`Ctrl+Shift+J`) — настройки AI-модели, правила проекта
- **`.cursorrules`** — файл в корне проекта для пользовательских инструкций AI

## Частые проблемы

| Проблема | Решение |
|---|---|
| Cursor не запускается | Проверьте, что установлена последняя версия |
| Не работает AI | Проверьте подключение к интернету и авторизацию |
| Не видит Git | Убедитесь, что Git установлен: `git --version` |
| Расширения VS Code | Большинство расширений VS Code совместимы с Cursor |

## Итого

Подключение к Cursor — это всего 3 шага:

1. **Скачать** и установить
2. **Войти** в аккаунт
3. **Открыть** папку с проектом

Никакой сложной настройки не требуется. Cursor работает "из коробки".
